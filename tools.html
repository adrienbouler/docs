Tools

PatchKit Tools is set of programs that can be used to access PatchKit functionality through command line.

Installation

PatchKit Tools require Ruby installation (also it must be added to the PATH variable so ruby command is available).

To install PatchKit Tools clone repository from GitHub and add it's location to the PATH variable.

Also you need to install required gems. Start with

gem install bundler

Then go to the repository src folder and perform

bundler install

Usage

Every tool is started by patchkit-tools command. The working directory is set to location from which script was called.

Syntax

patchkit-tools [tool_name] [tool_arguments]

Displaying help and tools list

patchkit-tools --help

Displaying specific tool help

patchkit-tools [tool_name] --help

Uploading application version with tools

You can use make-version tool to update your application version. It has huge advantage for uploading new versions for applications which already has got at least one version. In this case instead of uploading whole content, tools will generate diff and upload it to PatchKit. Note that the only difference is method of uploading. Result will be the same as uploading whole content.

Type patchkit-tools make-version --help to learn how to use it.
