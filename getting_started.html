Getting Started

PatchKit is currently in beta stage. If you’re willing to get a free account, please fill in the registration form.

To get started, please sign in to your PatchKit account.

Create the application

First, you have to create an application. In order to do so:

  1. Go to Applications page.

  2. Click on New Application button.

  3. Fill-in your application Name and select target Platform. If your game supports multiple platforms, you have to create an Application for each platform.

  4. Click on the Create button.

Your first application is now created!

When your application is created, you will be redirected to application dashboard. Here you can change it’s name, platform, get its secret and manage versions.

Upload your game

To upload your game you have to create a new version.

  1. Click on the New Version button.

  2. Type in your version Label. Versions initially are identified by id number (1, 2, 3 etc.), so label is something more human-like.

  3. Optionally specify the list of Changes.

  4. Click Upload Content button and select your local game’s content zip file. It should include all the files needed to launch the game.

NOTE! When making a zip file, make sure your game executable file will be located in the root of your zip file, not in a subdirectory!

Your web browser will upload the zip file, so you will have to wait until it’s done. When finished, you will be taken to Version Processing page. This page will display a real-time processing progress of your version. You can leave it open or get back to it later.

Uploading the new versions can also be done using PatchKit Tools.

Publishing your game

When your version has been successfully processed, you will be taken to Version Dashboard. Please read it carefully, since it will let you know if there’s anything wrong with the content you’ve uploaded.

In order to publish your version, click on the Publish button.

You will be asked if you’re 100% sure you want to do that, because that process cannot be reversed. When confirmed, your game is now published.

Downloading your game

Now, it’s time to try it out! On your Application dashboard, you’ll find Get Launcher button.

Click on it and it will download a small launcher executable.

You can now freely share this executable with anyone.
