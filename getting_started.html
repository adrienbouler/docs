Getting Started

PatchKit is currently in beta stage. If you’re willing to get a free account, please fill in the registration form.

To get started first please sign in to your PatchKit account.

Create the application

First you have to create an application. In order to do so:

  1. Go to Applications page.
  2. Click on Create New Application button.
  3. Fill-in your application Name and select target Platform. Note that if your game have version for multiple platform, you have to create multiple Applications.
  4. Click on the Create App button.

Your first application is now created!

Upload your game

When your application is created, you will be redirected to application dashboard. Here you can change it’s name, platform, get its secret and manage versions. To upload your game you have to create a new version.

  1. Click on the Create New Version button.
  2. Set your version Label. Versions initially are identified by id number (1, 2, 3 etc.), so label is something more human-like.
  3. Choose your game’s content zip file. It should include all the files needed to launch the game.
  4. Optionally specify the list of Changes.
  5. Click on the Upload & Create button.

Be Cautious! When making a zip file, make sure that your game executable file will be located in the root of your zip file, not in a subdirectory!

Now you have to be patient. First your web browser will upload the zip file, so you will have to wait until it is done. When finished, you will be taken into Version Processing page. This page will display a real-time processing progress of your version. You can leave it open (it will reload to Version Dashboard when finished) or get back to it later.

Publishing your game

When your version has been successfully processed, you should be taken into Version Dashboard. Please read it carefully because it will let you know if there’s something wrong with the content that you’ve uploaded.

In order to publish your version, click on the Publish Now button. You will be asked if you’re 100% sure that you want to do that because that process cannot be reversed. When confirmed, your game is now published.

Downloading your game

Now it’s time to try it out! On your Application Dashboard (if you’re lost go to Applications page and then click on your application name) you fill find Launcher frame with the link to download it. Click on it and it will download a small launcher executable. Try to run it and you will discover that shortly after your game is now installed and can be played!

You can freely distribute this URL to anyone that you want to give your game to. It will always point to the newest version of your game.
