Custom patcher (Unity3D)

You can create your custom patcher by modyfing sources of patcher made in Unity3D which sources can be found here.

Making modifications

Scripts

All of the patcher scripts are located in Scripts directory. Their functionality is based on PatchKit Unity Integration which, among other things, provides an easy connection with PatchKit API. Classes are split into a few modules:

  • PatchKit.Unity.Web - web functionality such as downloading files via HTTP or torrent
  • PatchKit.Unity.Patcher - core functionality of patcher application
  • PatchKit.Unity.Patcher.UI - updating user interface of patcher application
  • PatchKit.Unity.Patcher.Utilities - additional patcher components such as borderless patcher window

Layout

Patcher layout is done in Unity UI. Here you can find a few tutorials about using Unity UI.

Setting custom patcher

Configuring for the first time

  • Create separate application on PatchKit which will be used by patcher (for example - My Game Patcher).
  • Upload the first version of your custom patcher (important - remember to include patcher.manifest file in the package - it's automaticaly generated while building the patcher with Unity3D).
  • Copy patcher application secret and paste it into "Custom Patcher" field in your application settings (for which you want to set a custom patcher).

Updating custom patcher

Updating custom patcher is the same process as updating every applciation - you just need to upload a new version to your patcher application and publish it - old patcher versions will be automaticaly replaced by the new version on next start of launcher application.
