Custom patcher (Unity3D)

You can create your custom patcher by modifying sources of patcher made in Unity3D. The sources can be found here.

Making modifications

Scripts

All of the patcher scripts are located in Scripts directory. Their functionality is based on PatchKit Unity Integration which, among other things, provides an easy connection with PatchKit API. Classes are split into a few modules:

  • PatchKit.Unity.Web - web functionality such as downloading files via HTTP or torrent
  • PatchKit.Unity.Patcher - core functionality of patcher application
  • PatchKit.Unity.Patcher.UI - updating user interface of patcher application
  • PatchKit.Unity.Patcher.Utilities - additional patcher components such as borderless patcher window

Layout

Patcher layout is done in Unity UI. Here you can find a few tutorials about using Unity UI.

Setting custom patcher

Configuring for the first time

  1. Create a new application and give it a name, for instance: My Patcher. As a platform, select Other.
  2. Build a custom patcher on a corresponding platform from Unity. Make sure patcher.manifest file is in the same build folder. (patcher.manifest is automatically generated after building the patcher with Unity3D)
  3. Zip the project.
  4. Upload the zip to My Patcher application.

  5. In My Patcher application settings copy the secret code on the right pane.

  6. Open the game settings (the one youâ€™ve made the custom Patcher for) and scroll down to see Custom Patcher Secret. Paste the patcher secret here.

Updating custom patcher

Updating custom patcher is the same process as updating every application - you just need to upload a new version to your patcher application and publish it. Old patcher versions will be automatically replaced by the new version on the next start of the launcher.
